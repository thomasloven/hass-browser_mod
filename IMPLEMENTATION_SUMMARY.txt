                              EXAMPLE CONFIGURATION

type: custom:quick-bar-plus-card
title: Quick Actions
show_search: true
placeholder: Search or select an action...
categories:
  - name: Navigation
    items:
      - label: Dashboard
        icon: mdi:view-dashboard
        navigation_path: /lovelace/0
      - label: Settings
        icon: mdi:cog
        navigation_path: /config/dashboard
        
  - name: Lights
    items:
      - label: Turn off all lights
        icon: mdi:lightbulb-off
        service: light.turn_off
        service_data:
          entity_id: all
          
  - name: Climate
    items:
      - label: Set Temperature
        icon: mdi:thermometer
        service: climate.set_temperature
        service_data:
          entity_id: climate.living_room
          temperature: 22

INSTALLATION:
1. Install/Update Browser Mod
2. Clear browser cache (SHIFT+F5)
3. Restart Home Assistant

ADDING THE CARD:
1. Open dashboard in edit mode
2. Click "+ Add Card"
3. Search for "Quick Bar Plus"
4. Configure categories and items
5. Save and enjoy!
