## User Interactions

### 1. Click Main Card
- Card displays with icon and title
- Click anywhere on card to open dialog

### 2. Search (Optional)
- Type in search box to filter items
- Real-time filtering across all categories
- Items not matching are hidden

### 3. Select Action
- Click any item to execute its action
- Navigation items: Navigate to specified path
- Service items: Call service with data
- Dialog closes automatically after action

### 4. Close Dialog
- Click backdrop (dark area around dialog)
- Click X button in header
- After executing an action

## Example Use Cases

### 1. Dashboard Navigator
```yaml
categories:
  - name: Dashboards
    items:
      - label: Home
      - label: Lights
      - label: Energy
      - label: Security
```

### 2. Scene Controller
```yaml
categories:
  - name: Scenes
    items:
      - label: Movie Time
      - label: Good Night
      - label: Wake Up
      - label: Away
```

### 3. Quick Actions
```yaml
categories:
  - name: All Lights
    items:
      - label: Turn On
      - label: Turn Off
  - name: Climate
    items:
      - label: Heating On
      - label: Cooling On
      - label: Off
```

### 4. Multi-Purpose
```yaml
categories:
  - name: Navigate
    items: [Dashboard, Settings, Energy]
  - name: Lights
    items: [All On, All Off, Bedroom]
  - name: Climate
    items: [Heat, Cool, Fan]
  - name: Security
    items: [Arm Away, Arm Home, Disarm]
```

## Design Principles

1. **Simplicity** - Easy to understand and use
2. **Flexibility** - Works for many use cases
3. **Consistency** - Follows Home Assistant patterns
4. **Accessibility** - Clear labels and icons
5. **Performance** - Fast loading and interaction
